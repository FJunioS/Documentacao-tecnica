O que é HTML?

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

"Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

Introdução

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da  Web. A marcação HTML inclui "elementos" especiais, como: 

  <head> , <title>, <body>, <header> , <footer> , <article> , <section> , <p> , <div> , <span>, <img>, <aside>, <audio>, <canvas> , <datalist> , <details> , <embed> , <nav> , <output>, <progress>, <video>, <ul>, <ol>, <li>

  e muitos outros.

Um elemento HTML é separado de outro texto em um documento por "tags", que consistem no nome do elemento entre "<" e ">". O nome de um elemento dentro de uma tag é insensível a maiúsculas e minúsculas. Isto é, pode ser escrito em maiúsculas, minúsculas ou um mistura. Por exemplo, a tag <title> pode ser escrita como <Title>, <TITLE> ou de qualquer outra forma.

Raíz principal ou Main Root
Elemento Descrição
<html> O elemento <html> do HTML é responsável por contar todo o documento HTML, todos os outros elementos são descendentes deste.
Metadados dos Documentos

Metadado contém informações sobre a página. Isso inclui informação sobre estilos, scripts e dados para ajudar os softwares (Motores de pesquisa, navegadores, etc.) que usam e renderizam a página. Metadados para estilos e scripts podem ser definidos dentro da pagina ou ligação para outro arquivo que contem as informações

Elemento Descrição
<base> O elemento <base> do HTML especifica a URL base para usar em todos as URL relacionadas no documento.
<head> O elemento <head> do HTML contem os metadados do documentos, como seu título, scripts e CSS
<link> O elemento de ligação com recurso externo < link> do HTML especifica relações entre o atual documento e um recurso externo. Este elemente é muito comum para ligar com o CSS, mas também pode ser usado para estabelecer os ícones dos sites (tanto favicon quando ícones para tela inicial e aplicativos em dispositivos móveis ) além de outras coisas.
<meta> O elemento <meta> do HTML representa os metadados que não podem ser representados por outros meta-relacionados elementos do HTML, como <base>, <link>, <script>, <style>, ou <title>.
<style> O elemento <style> do HTML contém informações de estilo para um documento, ou parte dele.
<title> O elemento <title> do HTML define o título do documento que é mostrado na barra de título do navegador ou em uma aba de uma página.
Seção de Conteúdo

Antes de irmos para a seção de conteúdo, temos que falar sobra raiz da seção: <body>

Elemento Descrição
<body> O elemento <body> do HTML representa o conteúdo de um documento HTML. Pode existir apenas um elemento <body> em todo o documento

Com o elemento body esclarecido, podemos dizer que os elementos da seção de conteudo te permite organizar o conteúdo do documento em partes lógicas.
Use os elementos da seleção para criar partes entre o conteúdo da página, incluindo o cabeçario (header) e rodapé (footer), e elementos para poder identificar seções do conteúdo.

Elemento Descrição
<adress> O elemento <base> do HTML especifica a URL base para usar em todos as URL relacionadas no documento.
<article> O elemento <head> do HTML representa uma composição de conteúdo próprio em um documento, página, aplicação, ou site, em que a intenção é de ser independentemente distribuível ou reusavel.
<aside> O elemento < aside> do HTML representa uma porção do documento em que o conteudo é indiretamente relacionado com o conteúdo principal do documento.
<footer> O elemento <footer> do HTML representa o rodapé do elemento mais perto da seção de conteudo ou seção da raiz Um rodapé normalmente contém informações sobre o autor da seção, dados de direitos autorais ou links relacionado com o documento.
<header> O elemento <header> do HTML representa uma introdução ao conteudo, normalmente um grupo de introdução ou ajuda de navegação. Pode conter alguns elementos do cabeçario, como logo, formulário de busca, nome do autor, e outros elementos.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6>. O elemento <h1> - <h6> do HTML representa seis níveis da seção do cabeçario. <h1> é o maior nível da seção, e <h6> é o menor.
<hgroup>. O elemento <hgroup> do HTML representa um cabeçariode vários níveis para uma seção de um documento. Ele junta o conjunto do <h1> até o <h2>
<main> O elemento <main> do HTML represente o conteúdo dominante do <body> de um documento. O conteudo principal da tag <main> consiste em falar sobre o assunto principal de um tópico, ou a funcionalidade principal de uma aplicação.
<nav> O elemento <nav> do HTML representa uma seção da página onde a proposta é prover links de navegação, tanto para o próprio documento quanto para outros. Exemplos comuns da seção de navegação são os menus, tabelas de conteudos ou índices.
<section> O elemento <section> do HTML representa uma seção independente - a qual não precisa ter uma semântica mais específica à ela.
